About

On these pages you can download OSM derived Data as Shapefiles.
In the reports section you find some reports about boundary data in OSM

This site is inspired by gadm.org, Nominatim borken polygon report and OpenStreetMap Data
You can find full source code to generate these reports on github



faq

My country/state/whatever does not show up in the reports
This can have many reasons:

  • my code is buggy
  • Data is from 08 May 2012. Perhaps now it would show up.
  • Your country/state/whatever or some of its parent polygons dont form a closed geometry
  • esoteric tagging is used that my code does not understand
